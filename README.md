библиотека mm нужна для списков в столбик, строчку. Как пользоваться?  
1.Импортировать файл в программу (import mm) 
2.Cоздать данные переменные : a,b,c,d,e,f,g,nomber_mm Зачем они нужны? все кромен nomber_mm это значения строчки в списке а nomber_mm отвечает будет нумерация(=1) или без(=0)  
3.Создать в строчку: line(a,b,c,d,e,f,g,nomber_mm) в  Столбик menu(a,b,c,d,e,f,g,nomber_mm)
